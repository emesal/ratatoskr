pub struct ModelRegistry {
    entries: HashMap<String, ModelMetadata>,
    presets: BTreeMap<String, BTreeMap<String, PresetEntry>>,
}
Expand description

Centralised model metadata registry.

Thread-safe reads are ensured by storing data in a HashMap keyed by model ID. For concurrent access (e.g., from EmbeddedGateway), wrap in Arc<RwLock<>>.
