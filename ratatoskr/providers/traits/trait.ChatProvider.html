Providers that override this enable parameter validation in the registry. Default: empty (legacy behaviour — no validation).

Fetch metadata for a model from this provider’s upstream API.

Returns ModelNotAvailable if the provider doesn’t recognise the model, or NotImplemented if the provider doesn’t support metadata fetching. The registry walks the fallback chain on either of these.
